# 自动保存规则

在与用户讨论《元英视角》播客项目时，以下内容需要**自动保存**，无需用户提醒：

## 必须自动保存的内容

### 1. 重要决策

- 内容策略调整
- 定位变更
- 受众定义变化
- 任何"我们决定..."的内容

保存位置: `archives/chat-logs/YYYY-MM-DD-主题.md`

### 2. 分析输出

- 受众分析
- 数据分析
- 脚本评估
- 任何结构化的分析内容

保存位置: `docs/` 或 `archives/`

### 3. 脚本版本

- 每个脚本的版本变更
- 版本评估反馈
- 确认/否决记录

保存位置: `archives/script-versions/`

### 4. 日常对话摘要

- 每次对话结束时，自动保存对话摘要
- 包含讨论的主题、关键问题、结论
- 即使是"小"的讨论也可能有价值

保存位置: `archives/chat-logs/YYYY-MM-DD-daily.md`

## 保存格式

每个保存的文件应包含:

- 创建日期和时间
- 触发原因
- 完整内容
- 相关决策
- 后续Action（如有）

## 执行方式

在输出重要分析或决策后，立即调用write_to_file保存，不等待用户提醒。
对话结束时，主动询问是否需要保存摘要，或直接保存。
