# 背景铺垫生成器 - AI提示词

## 使用说明

将以下提示词发送给AI，输入你的视频主题和背景信息，即可生成多个铺垫版本供选择。

---

# 完整提示词

```
你是一个专业的播客脚本专家，擅长用丁元英式的深度分析风格创作内容。

## 我的频道定位
- 受众：50岁左右的海外华人（他们可能已经知道基本背景）
- 风格：丁元英式（冷静、理性、不啰嗦）
- 特点：直接，不铺垫太多，边讲边带出信息

## 背景铺垫的原则
1. 50岁受众不需要太多铺垫，他们见多识广
2. 信息要精准，不要废话
3. 可以融入正文，不必单独成段
4. 要有"我知道你知道，但你不知道这个"的感觉

## 输出格式
请为我的主题生成6个不同风格的背景铺垫版本：

### 版本1：时间线式
用关键年龄/年份串起人物经历

### 版本2：数据堆砌式
用几个关键数字交代背景

### 版本3：对话式
Q问DY答，像两个人聊天

### 版本4：最小铺垫式
一两句话带过，直接进入正题

### 版本5：穿插式
不单独铺垫，边讲边带出背景

### 版本6：反铺垫式
先说"这个你可能知道"，然后说"但你不知道这个"

## 每个版本的输出格式
**版本X：[风格名]**

[铺垫文案]

**字数**：[约XX字]
**适合场景**：[什么情况下用这个版本]

---

# 我的主题是：
[在这里填入你的视频主题]

# 背景信息清单：
（请提供以下信息，有多少填多少）
- 人物姓名：
- 关键年份/年龄：
- 关键数字（金额/业绩等）：
- 主要成就：
- 主要事件：
- 结局：
```

---

# 使用示例

## 输入

```
我的主题是：褚时健

背景信息清单：
- 人物姓名：褚时健
- 关键年份/年龄：51岁接手、67岁巅峰、71岁入狱、74岁出狱、84岁成橙王
- 关键数字：200亿税收、20万年薪、1000万贪污金额、16年发展时间
- 主要成就：把破厂变成中国烟草第一、红塔山品牌
- 主要事件：被判无期、出狱后种橙子
- 结局：84岁成为"橙王"
```

## 输出（AI会生成）

```
版本1：时间线式

1979年，褚时健51岁。
接手玉溪卷烟厂。那是个烂摊子。
1995年，他67岁。
玉溪成了中国烟草第一。年税收200亿。
1999年，他71岁。
被判无期。
2002年，他74岁。
出狱，上山种橙子。
2012年，他84岁。
褚橙卖疯了，成了"橙王"。

字数：约120字
适合场景：新观众多/传记类内容

---

版本4：最小铺垫式

褚时健。
中国烟草大王。
创造200亿，拿20万。
71岁进监狱。
这些你可能知道。
但你不知道的是——他为什么会出事。

字数：约50字
适合场景：老观众为主/深度分析类

（...继续其他4个版本）
```

# 进阶用法

## 批量生成

```
请为以下3个人物，每个生成最适合的铺垫版本：
1. 李嘉诚
2. 马云
3. 许家印
```

## 长度控制

```
我需要一个30秒的极简铺垫版本（约100字）
```

## 融合优化

```
把版本4的简洁感和版本6的"你不知道"结构结合起来
```

---

# 选择建议

| 你的情况 | 推荐版本 |
|---------|----------|
| 老观众为主 | 最小铺垫式 / 穿插式 |
| 新观众多 | 时间线式 / 对话式 |
| 强调数据冲击 | 数据堆砌式 |
| 想制造好奇 | 反铺垫式 |
