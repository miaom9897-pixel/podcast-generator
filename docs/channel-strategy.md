## 一、频道定位

### 1.1 一句话定位

> **用丁元英的视角，帮海外华人看清中国真相，做出资产和家庭的判断。**

### 1.2 核心价值主张

| 维度 | 内容 |
|------|------|
| **我们提供什么** | 墙内听不到的真话、深度的政策解读、理性的经济分析 |
| **受众获得什么** | 信息差、判断力、行动方向 |
| **为什么是我们** | 丁元英人设+AI能力+海外视角 |

### 1.3 品牌关键词

- **理性**：不煽情、不贩卖焦虑、用数据说话
- **真实**：说墙内不让说的话、揭示官方不会说的真相
- **深度**：不做表面文章、层层拆解本质
- **实用**：给受众具体的思考方向和行动建议

### 1.4 频道口号候选

1. "说墙内不让说的话"
2. "帮你看清，帮你判断"
3. "用丁元英的视角，看懂中国"
4. "海外华人的信息差"

## 二、目标受众深度分析

### 2.1 核心受众画像

| 维度 | 描述 |
|------|------|
| **年龄** | 45-65岁 |
| **地理** | 美国、加拿大、澳洲、新西兰、欧洲、东南亚华人社区 |
| **语言** | 中文为主，能理解普通话 |
| **身份** | 第一代移民、留学后留下、投资移民、技术移民 |
| **经济状况** | 中产及以上，有一定资产积累 |
| **教育背景** | 本科及以上居多 |
| **职业** | 专业人士、企业主、退休人员 |

### 2.2 受众核心关切（痛点）

#### 2.2.1 资产相关

| 痛点 | 具体表现 |
|------|----------|
| **国内房产** | 要不要卖？能不能卖？卖了钱怎么出来？ |
| **人民币存款** | 会不会贬值？要不要换成美元？ |
| **国内投资** | 股票、基金、理财产品还安全吗？ |
| **资金出境** | 外汇管制会不会更严？以后还能汇钱吗？ |

#### 2.2.2 家庭相关

| 痛点 | 具体表现 |
|------|----------|
| **父母养老** | 在国内的父母怎么办？养老金够不够？医疗怎么解决？ |
| **子女教育** | 要不要送孩子回国读书？中国教育还有优势吗？ |
| **家庭团聚** | 父母要不要接出来？签证怎么办？ |
| **遗产继承** | 国内的房子、存款以后怎么继承？ |

#### 2.2.3 身份相关

| 痛点 | 具体表现 |
|------|----------|
| **中国身份** | 要不要保留中国国籍？退籍有什么影响？ |
| **回国问题** | 以后还能不能回国？回去会不会有麻烦？ |
| **中美关系** | 如果中美关系恶化，对我有什么影响？ |

#### 2.2.4 信息需求

| 痛点 | 具体表现 |
|------|----------|
| **信息不对称** | 墙内新闻看不到真相，墙外新闻有偏见，不知道该信谁 |
| **判断困难** | 信息太多太杂，不知道怎么判断 |
| **决策焦虑** | 想做决定但不知道对不对，怕后悔 |

### 2.3 受众心理特征

| 特征 | 描述 |
|------|------|
| **怀旧又警惕** | 对中国有感情，但对政策保持警惕 |
| **务实理性** | 不喜欢空谈，喜欢有干货的内容 |
| **愿意深度思考** | 愿意花15-25分钟听深度分析 |
| **重视家庭** | 最关心的是家人的安全和福祉 |
| **有判断力** | 不盲从，会独立思考，但需要信息输入 |

### 2.4 受众消费习惯

| 维度 | 描述 |
|------|------|
| **收看时间** | 晚饭后、周末、通勤路上 |
| **设备** | 电视大屏、电脑、平板居多，手机较少 |
| **时长偏好** | 15-30分钟的深度内容 |
| **互动习惯** | 愿意留言评论，分享给朋友 |
| **付费意愿** | 有一定付费意愿，但需要建立信任 |

### 2.5 为什么不做年轻人（30岁以下）？

| 原因 | 解释 |
|------|------|
| **没有资产** | 30岁以下多数没有国内房产、大额存款，痛点不强 |
| **不关心政策** | 年轻人更关心职业发展、娱乐，对政策不敏感 |
| **消费习惯不同** | 喜欢短视频、快节奏，不愿意看长内容 |
| **变现困难** | 年轻人付费意愿和能力都较低 |
| **竞争激烈** | 做年轻人市场的频道太多了 |

### 2.6 为什么选择45-65岁？

| 原因 | 解释 |
|------|------|
| **有资产** | 这个年龄段大多有房产、存款、投资，对资产安全非常敏感 |
| **有牵挂** | 父母在国内、子女可能在海外，两边都有牵挂 |
| **有阅历** | 经历过很多事，能理解深度内容，有判断力 |
| **有时间** | 事业稳定或已退休，有时间看长内容 |
| **有付费能力** | 经济基础好，愿意为有价值的内容付费 |
| **竞争较少** | 专门做这个年龄段的中文频道不多 |

## 三、内容垂直度策略

### 3.1 "垂直"的真正含义

**常见误解**：垂直 = 只做一种话题

**正确理解**：垂直 = 所有内容服务于同一个受众、同一个价值主张

### 3.2 本频道的垂直逻辑

```
核心受众：45-65岁海外华人
核心价值：帮他们看清真相、做出判断
    ↓
所有内容都要回答一个问题：
"这件事跟我的资产、我的家人、我的未来有什么关系？"
```

### 3.3 内容分层结构

#### 第一层：主赛道（60%的内容）

**政策解读 + 经济分析**

| 子类型 | 说明 | 示例选题 |
|--------|------|----------|
| **重大政策** | 中央政策、法规变化 | 海南封关、延迟退休、房产税 |
| **经济事件** | 企业动态、市场变化 | 索尼撤厂、恒大暴雷、人民币汇率 |
| **中美关系** | 两国关系变化对普通人的影响 | 芯片禁令、签证政策、脱钩影响 |
| **社会热点** | 引发广泛讨论的社会事件 | 35岁职场、医保改革、养老金亏空 |

**为什么是主赛道**：

- 时效性强，容易蹭热度
- 信息差大，墙内看不到
- 受众刚需，直接影响他们的判断
- 容易产出，有事件就有内容

#### 第二层：副赛道（30%的内容）

**资产 + 养老 + 移民**

| 子类型 | 说明 | 示例选题 |
|--------|------|----------|
| **资产配置** | 国内资产怎么处理 | 国内房子要不要卖、人民币要不要换 |
| **养老规划** | 父母和自己的养老 | 国内养老金够不够、要不要接父母出来 |
| **移民润学** | 身份和出路问题 | 各国移民政策、润的时机和方式 |
| **财富传承** | 遗产和继承问题 | 国内房产继承、跨境遗产税 |

**为什么是副赛道**：

- 受众核心痛点，但不需要高频更新
- 可以结合主赛道做，比如"海南封关对资产配置的影响"
- 常青内容，可以长期获得流量

#### 第三层：调味品（10%的内容）

**人性 + 历史 + 哲学**

| 子类型 | 说明 | 示例选题 |
|--------|------|----------|
| **历史镜像** | 用历史预判未来 | 日本失去三十年、苏联解体启示 |
| **丁元英思维** | 强势文化、弱势文化 | 丁元英的投资哲学、看问题的方法 |
| **人性洞察** | 深层次的人性分析 | 为什么韭菜总被割、信息茧房 |

**为什么是调味品**：

- 提升内容深度和情感共鸣
- 建立人设差异化
- 不能当主菜，因为没有时效性和话题性

### 3.4 内容禁区（不做的内容）

| 类型 | 为什么不做 |
|------|----------|
| **纯搞钱/理财** | 和定位不符，容易变成理财博主 |
| **纯鸡汤/励志** | 没有信息量，受众不买账 |
| **娱乐八卦** | 完全不相关 |
| **国内时政键政** | 太敏感，容易翻车 |
| **美国国内政治** | 不是核心受众关心的 |
| **短视频** | 不是我们的优势 |
| **带货/推销** | 损害信任 |

### 3.5 如何保持垂直但不单调？

**方法一：同一事件多角度**

比如"海南封关"这个事件：

- 第一期：政策解读（是什么、为什么）
- 第二期：对资产的影响（要不要去海南投资）
- 第三期：历史对比（和香港、深圳的对比）

**方法二：不同事件同一逻辑**

比如"外资撤离"这个主题：

- 索尼篇：日韩企业为什么跑
- 苹果篇：美资企业为什么跑
- 总结篇：产业转移对中国的影响

**方法三：主赛道+副赛道结合**

比如"延迟退休"政策出台：

- 先做政策解读（主赛道）
- 再做"父母养老怎么规划"（副赛道）

## 四、选题系统

### 4.1 选题来源

| 来源 | 说明 | 操作方法 |
|------|------|----------|
| **新闻热点** | 刚发生的重大事件 | 每天浏览财新、华尔街日报、路透社 |
| **政策发布** | 国务院、央行、发改委文件 | 关注政府网站、政策解读公众号 |
| **数据发布** | 经济数据、统计公报 | 关注统计局、央行数据发布 |
| **受众反馈** | 评论区问题、私信 | 每期收集高频问题 |
| **竞品分析** | 其他频道在做什么 | 定期浏览竞品频道 |
| **长期规划** | 提前规划的系列内容 | 按季度规划内容日历 |

### 4.2 选题评估矩阵

每个选题从四个维度打分（1-5分）：

| 维度 | 说明 | 权重 |
|------|------|------|
| **时效性** | 是否正在发生、是否有热度 | 30% |
| **信息差** | 墙内能不能看到、受众知不知道 | 25% |
| **痛点强度** | 和受众的钱、家人、未来的关系 | 25% |
| **可延展性** | 能不能讲15分钟以上的深度 | 20% |

**评分标准**：

- 16分以上：立即做
- 12-15分：可以做
- 8-11分：考虑做
- 8分以下：放弃

### 4.3 选题示例评估

**示例一：索尼惠州关厂**

| 维度 | 分数 | 理由 |
|------|------|------|
| 时效性 | 5 | 刚发生 |
| 信息差 | 4 | 墙内有报道但分析不深 |
| 痛点强度 | 4 | 和制造业转移、经济走势相关 |
| 可延展性 | 5 | 可以讲产业链、日本教训、对受众影响 |
| **总分** | **18** | **立即做** |

**示例二：某明星离婚**

| 维度 | 分数 | 理由 |
|------|------|------|
| 时效性 | 5 | 热搜第一 |
| 信息差 | 1 | 到处都是 |
| 痛点强度 | 1 | 和受众无关 |
| 可延展性 | 2 | 没什么深度可讲 |
| **总分** | **9** | **放弃** |

### 4.4 选题日历模板

| 周次 | 主赛道（必做） | 副赛道（选做） | 调味品（穿插） |
|------|---------------|---------------|---------------|
| 第1周 | 索尼撤厂+海南封关 | - | - |
| 第2周 | 中央经济工作会议 | 2025年资产配置建议 | - |
| 第3周 | 人民币汇率破7.3 | - | 日本失去三十年 |
| 第4周 | 延迟退休政策 | 父母养老规划 | - |

### 4.5 热点响应机制

| 热点级别 | 响应时间 | 处理方式 |
|----------|----------|----------|
| **S级**（重大政策、突发事件） | 24-48小时内 | 加急制作，抢时效 |
| **A级**（重要事件、持续热点） | 一周内 | 正常排期，保证质量 |
| **B级**（一般事件） | 两周内 | 可做可不做，看排期 |
| **C级**（小事件） | 不响应 | 放弃 |

## 五、内容结构框架

### 5.1 标准脚本结构

```
【开场钩子】1-2分钟
- 抛出事件/问题
- 制造悬念
- 给出核心观点预告

【背景铺垫】2-3分钟
- 事件是什么
- 官方怎么说
- 为什么要关注

【深度拆解】8-12分钟
- Layer 1：表面现象
- Layer 2：背后原因
- Layer 3：本质逻辑
- Layer 4：历史对比（可选）

【受众影响】3-5分钟
- 这跟海外华人有什么关系
- 对资产的影响
- 对家人的影响

【行动建议】2-3分钟
- 该怎么想
- 该怎么做
- 该关注什么

【互动引导】1-2分钟
- 提出问题
- 引导评论
- 收尾金句

【结尾】30秒
- 总结一句话
- 告别语
```

### 5.2 时长控制

| 类型 | 建议时长 | 适用场景 |
|------|----------|----------|
| **深度分析** | 18-25分钟 | 主赛道内容，重大事件 |
| **快速解读** | 10-15分钟 | 热点响应，时效优先 |
| **系列内容** | 每期15-20分钟 | 多期连载，话题较大 |

### 5.3 金句分布

每期内容应该有5-8个金句，分布在：

- 开场1个（抓住注意力）
- 每个Layer结尾1个（强化记忆点）
- 结尾1-2个（让人回味）

## 六、人设与风格指南

### 6.1 丁元英人设特征

| 特征 | 表现方式 |
|------|----------|
| **冷酷理性** | 不煽情、不愤怒、就事论事 |
| **洞察本质** | 穿透表象、找到规律 |
| **不卑不亢** | 不谄媚权贵、不迎合大众 |
| **有底线** | 知道什么该说什么不该说 |
| **悲悯但不滥情** | 关心普通人，但不卖惨 |

### 6.2 语言风格

| 要 | 不要 |
|----|------|
| 简短有力的句子 | 长句、复杂句 |
| 口语化表达 | 书面语、官话 |
| 数据和案例 | 空洞的形容词 |
| 提问和反问 | 单向灌输 |
| 停顿和节奏 | 一口气讲完 |

### 6.3 禁用词汇

| 禁用 | 原因 |
|------|------|
| "相信国家""相信政府" | 不符合定位 |
| "一定会""肯定会" | 太绝对 |
| "赶紧买""赶紧卖" | 构成投资建议 |
| 人身攻击的词汇 | 有法律风险 |
| 过于敏感的政治表态 | 有风险 |

### 6.4 DY与Q的角色分工

| 角色 | 功能 |
|------|------|
| **DY** | 主讲、输出观点、提供分析、给出金句 |
| **Q** | 提问、代表普通观众的疑惑、推动对话、适当质疑 |

**Q的提问类型**：

- 澄清型："什么意思？"
- 质疑型："但是不是也有另一种可能？"
- 深入型："为什么会这样？"
- 关联型："这和我们有什么关系？"
- 行动型："那我们该怎么办？"

## 七、竞品分析与差异化

### 7.1 主要竞品

| 频道类型 | 代表频道 | 特点 |
|----------|----------|------|
| **政治评论型** | 文昭、江峰 | 强政治立场、键政为主 |
| **财经分析型** | 王剑 | 专业财经分析 |
| **移民资讯型** | 各类移民频道 | 移民政策、生活攻略 |
| **生活分享型** | 海外华人Vlog | 生活日常、文化差异 |

### 7.2 我们的差异化

| 维度 | 竞品 | 我们 |
|------|------|------|
| **立场** | 强政治倾向 | 相对中立理性 |
| **风格** | 愤怒、激动 | 冷静、深度 |
| **受众** | 各年龄段 | 聚焦45-65岁 |
| **形式** | 单人评论 | 双人对话 |
| **人设** | 自己 | 丁元英 |
| **重点** | 评论事件 | 分析影响、给建议 |

### 7.3 核心差异化点

> **我们不是在骂政府，我们是在帮你判断。**

不做：纯粹的政治批评、情绪宣泄
只做：理性分析、影响评估、行动建议

## 八、运营节奏

### 8.1 更新频率

| 频率 | 说明 |
|------|------|
| **常规** | 每周1-2期 |
| **热点加更** | 有重大事件时加更 |
| **休息** | 每月可休息1周 |

### 8.2 制作周期

| 阶段 | 时间 | 产出 |
|------|------|------|
| 选题确定 | D-3 | 选题+大纲 |
| 脚本撰写 | D-2 | 完整脚本 |
| 语音生成 | D-1 | 音频文件 |
| 封面+剪辑 | D-1 | 视频成品 |
| 发布 | D | 上线 |

### 8.3 发布时间

| 时区 | 最佳发布时间 |
|------|-------------|
| 美西 | 周五/周六晚上8点 |
| 美东 | 周五/周六晚上8点 |
| 澳洲 | 周六/周日下午 |

**原则**：周末发布，因为受众周末更有时间看长内容。

## 十、风险管理

### 10.1 内容风险

| 风险 | 应对方式 |
|------|----------|
| **政治敏感** | 不点名攻击领导人、不煽动、不造谣 |
| **法律风险** | 加免责声明、不构成投资建议 |
| **平台风险** | 不用敏感词、不放敏感画面 |

### 10.2 免责声明模板

```
⚠️ 免责声明：
本频道为AI演绎丁元英思维，所有观点仅供参考讨论。
不代表任何政治立场，不构成投资建议。

```

## 附录：核心问题快速参考

### Q1：要不要垂直？

**A**：要，但垂直是围绕同一个受众和价值，不是只做一种话题。

### Q2：政策+经济+资产混着发可以吗？

**A**：可以，因为都服务于"帮海外华人做判断"这个核心。

### Q3：搞钱/人性内容可以做吗？

**A**：可以穿插，但不能当主菜，最多占10%。

### Q4：要不要做年轻人？

**A**：不要，45-65岁是核心受众，不要为了流量分散定位。

### Q5：一期视频多长合适？

**A**：15-25分钟，这是目标受众能接受的深度内容时长。

### Q6：更新频率多少合适？

**A**：每周1-2期，有热点可加更，保证质量优先。
