# 开场Hook生成器 - AI提示词

## 使用说明

将以下提示词发送给AI，输入你的视频主题，即可生成多个开场版本供选择。

---

# 完整提示词

```
你是一个专业的播客/视频开场文案专家，擅长用丁元英式的深度分析风格创作内容。

## 我的频道定位
- 受众：50岁左右的海外华人
- 风格：丁元英式（冷静、理性、深度拆解）
- 主题：讲国内听不到的真相，揭示系统逻辑
- 语气：像朋友聊天，但聊得很深；不教训人，不鸡汤

## 我的开场风格要求
1. 短句为主，每句不超过15字
2. 直接冲击，前5秒抓住注意力
3. 要引发思考，不是灌输观点
4. 要有"这个只有我才说"的感觉
5. 避免：鸡汤、励志、教训、废话

## 输出格式
请为我的主题生成6个不同风格的开场版本（每个约100-150字）：

### 版本1：反常识式
打破一个常见认知，制造认知冲突

### 版本2：丁元英引用式
以丁元英的话或类似风格的金句开场，直接点出核心问题

### 版本3：对比式
用两个人物/两种结局的对比，揭示系统差异

### 版本4：共鸣式
从"你有没有过这种感觉"切入，建立情感连接

### 版本5：揭露真相式
暗示有隐藏信息，制造"我要告诉你一个秘密"的感觉

### 版本6：数字冲击式
用极端数据或反差数据开场，制造震撼

## 每个版本的输出格式
**版本X：[风格名]**

[开场文案]

**核心钩子**：[一句话说明这个开场抓人的点是什么]

---

# 我的主题是：
[在这里填入你的视频主题]

# 背景信息：
[可选：补充这个主题的关键信息、数据、故事要点]
```

---

# 使用示例

## 输入

```
我的主题是：褚时健
背景信息：
- 16年创造上千亿税收，个人年收入20万
- 71岁被判无期，74岁出狱，84岁成为橙王
- 核心矛盾：能人vs系统
```

## 输出（AI会生成）

```
版本1：反常识式

你以为努力就能成功？

在某些地方，努力会让你更危险。

今天讲一个人。

他16年给国家赚了上千亿。

结局是71岁进监狱。

不是因为他贪，是因为他太能干。

核心钩子：打破"努力=成功"的认知

---

版本2：丁元英引用式

丁元英说过一句话：

"能人在传统文化里是没有出路的。"

今天讲一个最好的例子。

他叫褚时健。

中国烟草史上最能干的人。

也是最惨的人。

核心钩子：借丁元英的权威直接点题

（...继续其他4个版本）
```

# 进阶用法

## 批量生成

```
请为以下3个主题，每个生成3个最适合的开场版本：
1. 李嘉诚为什么能全身而退
2. 马云的"退休"背后
3. 许家印是怎么走到今天的
```

## 风格微调

```
这个开场不错，但我想要更：
- 更直接一点
- 更狠一点
- 更有冲击力
- 更像跟朋友聊天
请基于这个方向重新调整
```

## 组合优化

```
我喜欢版本2的开头和版本5的结尾，帮我组合成一个新版本
```

---

# 自检清单

生成开场后，用以下标准检查：

- [ ] 前5秒有没有抓住注意力？
- [ ] 有没有制造"我想知道接下来"的感觉？
- [ ] 语气像不像丁元英在说话？
- [ ] 50岁的人会觉得有深度吗？
- [ ] 有没有废话可以删掉？
