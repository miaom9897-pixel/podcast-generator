# 数据与信息来源指南

## 📋 核心原则

- 信息要可靠，要能查证
- 数据要有出处，不能瞎编
- 优先使用官方/权威来源

---

# 六、信息核实方法

## 6.1 交叉验证

```
规则：重要数据至少要有2个独立来源确认
```

| 步骤 | 方法 |
|------|------|
| 1 | 找到原始数据来源 |
| 2 | 用第二个来源核实 |
| 3 | 检查时间是否一致 |
| 4 | 检查统计口径是否相同 |

## 6.2 识别虚假信息

| 危险信号 | 说明 |
|----------|------|
| 无具体来源 | "据说"、"有人说" |
| 极端数字 | 太夸张的数字要查证 |
| 情绪化表达 | 愤怒/恐惧导向的内容 |
| 二手传播 | 截图没有原文链接 |

## 6.3 引用格式

```
在脚本中提到数据时：

✅ 正确："根据国家统计局的数据，2023年GDP增速是5.2%"
✅ 正确："财新网报道..."
✅ 正确："彭博社的数据显示..."

❌ 错误："有数据显示..."
❌ 错误："大家都知道..."
```

# 七、各主题常用数据来源速查

## 资本往事（人物传记）

| 需要 | 来源 |
|------|------|
| 人物背景 | 百度百科、Wikipedia |
| 企业数据 | 企查查、上市公司年报 |
| 时间线 | 新闻报道汇总 |
| 争议事件 | 财新、调查报道 |

## 历史镜像（历史事件）

| 需要 | 来源 |
|------|------|
| 历史数据 | FRED、世界银行 |
| 事件背景 | 学术论文、书籍 |
| 当时报道 | 历史新闻档案 |
| 后续影响 | 研究报告 |

## 时政热点

| 需要 | 来源 |
|------|------|
| 政策原文 | 政府网站 |
| 专家解读 | 主流财经媒体 |

| 国际反应 | 外媒报道 |
