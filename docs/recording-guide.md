# 二、录音环境设置

## 2.1 房间处理

| 问题 | 解决方案 |
|------|----------|
| **回声大** | 挂窗帘、铺地毯、加吸音棉 |
| **杂音多** | 关窗、关空调、选安静时段 |
| **混响重** | 用衣柜/书架做天然吸音 |

## 2.2 录音位置

```
最佳位置：
- 离麦克风10-20cm
- 嘴巴正对麦克风
- 略微偏离中心（减少爆破音）

避免：
- 对着窗户/墙壁（反射）
- 空旷房间中间
- 空调出风口附近
```

# 四、录音技巧

## 4.1 语速控制

| 内容 | 建议语速 | 说明 |
|------|----------|------|
| 开场Hook | 快（每分钟200字）| 抓注意力 |
| 核心真相 | 中快（每分钟180字）| 保持节奏 |
| 拆解分析 | 中慢（每分钟150字）| 让人跟上 |
| 金句 | 慢（每分钟120字）| 强调 |
| 收尾 | 中（每分钟160字）| 自然结束 |

## 4.2 停顿技巧

| 位置 | 停顿时长 |
|------|----------|
| 金句前 | 0.5-1秒 |
| 金句后 | 1-2秒 |
| 模块切换 | 2-3秒 |
| 提问后 | 1秒 |
| 换气 | 自然（0.3秒）|

## 4.3 情绪控制

| 内容 | 情绪 | 技巧 |
|------|------|------|
| 冲击性内容 | 平静 | 越狠的话越平静地说 |
| 金句 | 略低沉 | 制造重量感 |
| Q的问题 | 好奇 | 语调上扬 |
| 收尾 | 温和 | 拉近距离 |

## 4.4 常见问题处理

| 问题 | 解决方案 |
|------|----------|
| **口水音** | 喝水、苹果汁（减少粘稠） |
| **爆破音（P/B/T）** | 偏离麦克风，加防喷罩 |
| **齿音（S/Z）** | 后期用De-esser处理 |
| **读错/说错** | 停顿2秒，重新说，后期剪掉 |
| **翻页声** | 用平板/电脑看稿，静音鼠标 |

# 五、后期制作流程

## 5.1 标准后期流程

```
1. 导入音频
   └── 检查轨道是否正确

2. 粗剪
   ├── 删除明显的错误
   ├── 删除长停顿（超过3秒的）
   └── 删除口误重录部分

3. 音频处理
   ├── 降噪（Noise Reduction）
   ├── 均衡化（EQ）
   ├── 压缩（Compressor）
   └── 限制器（Limiter）

4. 精剪
   ├── 调整节奏
   ├── 缩短过长的停顿
   └── 添加音乐/音效（可选）

5. 响度标准化
   └── 调整到 -14 LUFS（YouTube标准）

6. 导出
   └── MP3 192kbps 或 AAC 256kbps
```

## 5.2 降噪步骤（Audacity）

```
1. 选择一段纯背景噪音（无人说话）
2. Effect → Noise Reduction → Get Noise Profile
3. 选择整段音频
4. Effect → Noise Reduction → 
   - Noise Reduction: 12dB
   - Sensitivity: 6
   - Frequency Smoothing: 3
5. 点击OK
```

## 5.3 EQ设置建议

| 频段 | 调整 | 作用 |
|------|------|------|
| 80Hz以下 | 切掉（High Pass） | 去除低频隆隆声 |
| 100-200Hz | 略减 | 去除闷声 |
| 2-4kHz | 略加 | 增加清晰度 |
| 8kHz以上 | 略减或不动 | 避免刺耳 |

## 5.4 压缩设置建议

```
Threshold: -18dB
Ratio: 3:1
Attack: 10ms
Release: 100ms
Makeup Gain: 根据需要调整
```

# 六、双人对话录音特殊技巧

## 6.1 Q角色录音技巧

| 技巧 | 说明 |
|------|------|
| 自然反应 | 不要读稿，要像真的在听 |
| 语气变化 | 困惑、追问、惊讶要有区别 |
| 不抢话 | 等DY说完再说 |
| 适度重叠 | 偶尔的"嗯"、"对"可以 |

## 6.2 对话节奏

```
DY说完 → 停顿0.5秒 → Q提问 → DY回答

不要：
- Q立刻接话（太假）
- 停顿太久（节奏断）
- 两人同时说话（后期难剪）
```

## 6.3 分段录制策略

```
建议分段：
- 开场（单独录）
- 每个真相+拆解（单独录）
- 意义延伸（单独录）
- 收尾（单独录）

好处：
- 每段可以多次录制
- 累了可以休息
- 后期更灵活
```
