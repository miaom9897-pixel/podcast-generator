# 三、每个模块的详细指南

## 模块1：开场Hook

**目的**：前5秒决定观众走不走

**已有资源**：

- 12种开场版本（A-L）
- 开场Hook生成器提示词

**时长**：1-2分钟

**关键点**：

- 第一句话必须抓人
- 不要说"大家好"、"欢迎来到"这种废话
- 直接进入冲击性内容

---

## 模块2：本期预告

**目的**：告诉观众会得到什么，值得听下去

**模板**：

```
今天讲[人物/事件]。
讲[X]个真相：
第一，[真相1核心]
第二，[真相2核心]
第三，[真相3核心]
听完你会明白[核心价值]。
```

**示例**：

```
今天讲褚时健。
讲5个真相：
第一，他是被设计好的牺牲品
第二，能干为什么是一种罪
第三，这不是个案，是系统
第四，出路在哪
第五，你的脑子润出来了吗
听完你会明白：问题不在你，在系统。
```

---

## 模块3：背景铺垫

**目的**：给新观众必要信息

**已有资源**：

- 6种铺垫风格
- 背景铺垫生成器提示词

**关键点**：

- 50岁观众不需要太多铺垫
- 边讲边带比单独铺垫好
- 可以用"你可能知道，但你不知道这个"结构

---

## 模块4-8：核心真相

**目的**：脚本核心，你的价值所在

**结构**：

```
【真相抛出】
一句话说清楚这个真相是什么

【证据支撑】
- 数据
- 案例
- 对比
- 引用

【Q问DY答】
穿插Q的提问，让节奏更活

【真相强化】
用金句收尾
```

**关键点**：

- 每个真相要有"别人不说"的感觉
- 要有证据，不能空口白话
- 短句为主，每句不超过15字
- 每个真相结尾要有金句

---

## 模块5-9：拆解

**目的**：分析"为什么"，这是丁元英风格的核心

**结构**：

```
【提问】
你问为什么？

【系统分析】
因为[系统层面的原因]

【规律总结】
这个规律是[XX]

【丁元英引用】（可选）
丁元英怎么说来着？"[语录]"

【回到案例】
[人物]就是这样
```

**关键点**：

- 不止说what，要说why
- 要上升到系统层面，不是就事论事
- 要揭示规律，不是描述现象

---

## 模块10：意义延伸

**目的**：告诉观众"跟你有什么关系"

**结构**：

```
【转向观众】
现在聊聊屏幕前的你

【身份定位】
如果你在海外...
如果你在体制里待过...

【三层关系】
1. 对你的资产意味着什么
2. 对你的家庭意味着什么
3. 对你的思维意味着什么

【自检问题】
问你几个问题，不用回答我：
1. [问题1]
2. [问题2]
3. [问题3]
```

---

## 模块11：收尾

**目的**：总结+行动+订阅

**结构**：

```
【一句话总结】
今天的核心是什么？就一句话：[核心观点]

【行动指引】
如果这期对你有启发，你可以：
1. [行动1]
2. [行动2]

【点赞订阅】
如果觉得有价值，点个赞。
让更多人看到。
我们下期见。
```

# 四、音频节奏控制

## 语速控制

| 模块 | 建议语速 | 原因 |
|------|----------|------|
| 开场Hook | ⚡快 | 抓注意力 |
| 真相抛出 | ⚡快 | 制造冲击 |
| 拆解分析 | 🐢慢 | 让观众跟上思路 |
| 金句 | 🐢慢+停顿 | 强调 |
| 收尾 | 🚶中等 | 自然结束 |

## 停顿控制

| 位置 | 停顿时长 |
|------|----------|
| 金句前 | 1秒 |
| 金句后 | 2秒 |
| 模块切换 | 2-3秒 |
| 提问后 | 1秒 |

## 音量控制

| 内容 | 音量 |
|------|------|
| 正常内容 | 中等 |
| 重点强调 | 略高 |
| 金句 | 略低+慢（制造重量感）|
| 提问 | 略高（引起注意）|

# 五、Q角色使用指南

## Q的作用

| 作用 | 说明 |
|------|------|
| 打断节奏 | 避免单口播客太闷 |
| 代表观众 | 问出观众心里的问题 |
| 推进内容 | 引出下一个话题 |
| 增加可听性 | 两个声音比一个好听 |

## Q的话术风格

| 类型 | 示例 |
|------|------|
| 困惑 | "等等，这是什么意思？" |
| 追问 | "那他为什么要这样做？" |
| 质疑 | "你是说他被人搞了？" |
| 总结 | "所以你的意思是..." |
| 连接 | "那这跟我们有什么关系？" |

## Q的分布

| 时间段 | Q出现频率 |
|--------|----------|
| 前15分钟 | 每3-5分钟1次 |
| 中间 | 每5-8分钟1次 |
| 后15分钟 | 减少，让DY收尾 |

# 七、金句分布

## 每期金句数量：8-12个

| 位置 | 数量 | 作用 |
|------|------|------|
| 开场 | 1个 | 抓住注意力 |
| 每个真相结尾 | 各1个 | 强化记忆点 |
| 每个拆解结尾 | 各1个 | 升华观点 |
| 意义延伸 | 1-2个 | 连接观众 |
| 收尾 | 1-2个 | 让人回味 |

## 金句写法

| 类型 | 示例 |
|------|------|
| 对比式 | "他创造了200亿，拿了20万" |
| 反常识 | "在某些地方，能干是一种罪" |
| 丁元英式 | "规律不会因为你不喜欢，就对你网开一面" |
| 总结式 | "问题不是你行不行，是你在什么位置" |
