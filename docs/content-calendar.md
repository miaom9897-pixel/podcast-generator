## 二、系列一：【资本往事】

### 2.1 系列定位

> **讲商业大佬的真实故事，用丁元英的视角揭示背后的规律。**

这不是歌功颂德，也不是落井下石，而是冷静地看一个人怎么起来、怎么倒下、规律是什么。

### 2.2 脚本结构模板

```
【开场】30秒
- 一句话定义这个人："如果要用一句话形容XXX，就是..."
- 结局预告："他的结局是..."

【第一幕：起点】3分钟
- 他从哪里来？
- 早年经历
- 第一桶金

【第二幕：崛起】4分钟
- 怎么做大的？
- 关键抉择
- 成功的核心原因

【第三幕：巅峰】2分钟
- 最风光的时刻
- 当时的他怎么想的？

【第四幕：转折】3分钟
- 什么时候开始出问题？
- 危机的种子是什么？
- 他有没有察觉？

【第五幕：结局】2分钟
- 最后怎么样了？
- 现在在哪里？

【元英点评】2分钟
- 规律是什么？
- 对我们有什么启示？
- 金句收尾

总时长：约15分钟
```

## 四、系列三：【元英说理】

### 4.1 系列定位

> **用丁元英的方式，讲人生的道理。**

不是心灵鸡汤，是冷静的分析、真实的规律。针对50岁以上人群的人生阶段和关切。

### 4.2 脚本结构模板

```
【开场】30秒
- 抛出一个问题或现象
- "今天我想聊一个话题..."

【正反两面】4分钟
- 大多数人怎么想的？
- 这种想法的问题是什么？
- 另一种角度是什么？

【规律】3分钟
- 底层规律是什么？
- 历史或案例佐证

【落地】2分钟
- 具体怎么做？
- 怎么判断？

【收尾】1分钟
- 金句
- 留问题让观众思考

总时长：约10分钟
```
