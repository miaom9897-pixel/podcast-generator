## 二、频道定位（最终确定版）

### 2.1 一句话定位

> **用丁元英的视角，帮海外华人看清中国真相，做出资产和家庭的判断。**

### 2.2 目标受众

| 维度 | 描述 |
|------|------|
| **年龄** | 50-65岁（核心） |
| **地理** | 美国、加拿大、澳洲、欧洲海外华人 |
| **身份** | 第一代移民 |
| **关切** | 国内资产、父母养老、政策变化 |

### 2.3 人设原则

> **思想百家，嘴巴丁元英**

- 思想：可以融合索罗斯、巴菲特、芒格、达利欧、塔勒布、纳瓦尔、鲁迅等
- 表达：必须用丁元英的句式、语气、节奏

## 三、内容策略（基于数据调整后）

### 3.1 内容比例

| 类型 | 比例 | 说明 |
|------|------|------|
| **时政热点分析** | 50% | 有热点就做，这是流量密码 |
| **资本往事/历史镜像** | 30% | 没热点时做，保持更新节奏 |
| **道理/人性** | 20%以下 | 穿插，不主打 |

### 3.2 更新逻辑

```
有热点 → 做时政（优先级最高）
没热点 → 做故事（保持更新节奏）
```

### 3.3 故事类内容策略

- 优先做**受众同龄人**的故事（褚时健、牟其中、知青一代）
- 大众人物（李嘉诚、马云）用于引流
- 混合搭配
