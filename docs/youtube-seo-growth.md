# 二、标题写法策略

## 2.1 标题公式

### 公式1：悬念式

```
[人物/事件] + [悬念点] + ？
```

示例：`褚时健：为什么创造200亿的人，只敢贪1000万？`

### 公式2：反常识式

```
[反常识观点]，[补充说明]
```

示例：`能干是一种罪？褚时健的真相`

### 公式3：揭露式

```
[人物/事件]：[别人不说的点]
```

示例：`褚时健：没人告诉你的5个真相`

### 公式4：数字冲击

```
[惊人数字] vs [对比数字]，[结论]
```

示例：`200亿 vs 20万：褚时健悲剧的真相`

### 公式5：时间紧迫

```
【深度】[人物/事件]：[核心价值]
```

示例：`【深度】褚时健：能人为什么没有好下场`

## 2.2 标题禁忌

| 禁忌 | 原因 | 替代 |
|------|------|------|
| 太长（>60字符）| 显示不全 | 控制在40-50字符 |
| 全大写 | 显得廉价 | 正常书写 |
| 过度标题党 | 点击后退出率高 | 夸张但真实 |
| 无悬念 | 没有点击欲望 | 加入问号或对比 |

## 2.3 标题模板库（褚时健系列）

| 类型 | 标题示例 |
|------|----------|
| 悬念式 | `褚时健：为什么能干是一种"罪"？` |
| 揭露式 | `褚时健：没人敢讲的5个真相` |
| 数字式 | `200亿vs20万：褚时健真正的悲剧` |
| 对比式 | `同样能干，为什么任正非成首富，褚时健进监狱？` |
| 提问式 | `褚时健进监狱，问题出在哪？不是你想的那样` |

# 五、缩略图设计

## 5.1 缩略图核心原则

| 原则 | 说明 |
|------|------|
| **高对比度** | 在小屏幕上要清晰 |
| **大文字** | 3-5个字，看得清 |
| **人脸** | 人脸能吸引注意力 |
| **情绪** | 震惊、疑问、严肃 |
| **品牌一致性** | 固定色调和风格 |

## 5.2 缩略图模板

### 风格1：人物+大字

```
左侧：人物照片（占2/3）
右侧：3-5个大字（核心概念）
底部：频道logo小角标
背景：深色渐变
```

### 风格2：数字冲击

```
上方：大数字（200亿）
中间：vs 符号
下方：对比数字（20万）
侧面：人物照片
```

### 风格3：问号悬念

```
人物照片 + 大问号（？）
底部：一行文字
红色/黄色强调色
```

## 5.3 缩略图文字建议

| 主题 | 缩略图文字 |
|------|-----------|
| 褚时健 | "能干是罪？" |
| 马云 | "神坛跌落" |
| 任正非 | "至暗时刻" |
| 李嘉诚 | "为何撤退" |

# 六、发布时间策略

## 6.1 最佳发布时间（海外华人）

| 地区 | 最佳时间（当地） | 对应北京时间 |
|------|------------------|--------------|
| 美西 | 周六/日上午10点 | 周日凌晨2点 |
| 美东 | 周六/日上午10点 | 周日晚上11点 |
| 澳洲 | 周六/日晚上8点 | 周六下午5点 |

## 6.2 综合建议

**北京时间周六下午4-6点发布**

理由：

- 美东周六早上（起床时间）
- 美西周五晚上（周末开始）
- 澳洲周六晚上（黄金时段）

# 七、从0到1万订阅增长策略

## 7.1 阶段一：0-1000（打基础）

| 重点 | 行动 |
|------|------|
| **确立定位** | 前10期内容风格统一 |
| **打磨内容** | 每期都要高质量 |
| **收集反馈** | 认真看评论，调整方向 |
| **固定更新** | 每周至少2期 |

**关键指标**：完播率>40%

## 7.2 阶段二：1000-5000（扩圈）

| 重点 | 行动 |
|------|------|
| **蹭热点** | 重大事件48小时内出内容 |
| **系列化** | 资本往事、历史镜像形成系列 |
| **互动** | 回复评论，建立社区感 |
| **跨平台** | 把内容同步到Spotify、小红书 |

**关键指标**：订阅转化率>3%

## 7.3 阶段三：5000-10000（突破）

| 重点 | 行动 |
|------|------|
| **爆款复制** | 分析高播放视频，复制模式 |
| **合作推广** | 找其他频道互推 |
| **付费推广** | YouTube广告投放测试 |
| **社区建设** | 会员、Discord、微信群 |

**关键指标**：月增长>20%

# 九、互动策略

## 9.1 评论区运营

| 时间 | 行动 |
|------|------|
| 发布后1小时 | 回复前5条评论 |
| 发布后24小时 | 回复所有有价值的评论 |
| 日常 | 每天查看新评论，及时回复 |

## 9.2 评论回复模板

| 评论类型 | 回复模板 |
|----------|----------|
| 赞同 | "谢谢支持，说到你心里了" |
| 补充 | "感谢补充，这个角度也很重要" |
| 质疑 | "你的观点也有道理，欢迎讨论" |
| 提问 | "好问题，下期可以专门聊聊这个" |

## 9.3 引导评论的话术

| 位置 | 话术 |
|------|------|
| 视频中间 | "你觉得他做的对吗？评论区聊聊" |
| 视频结尾 | "这期讲的，你有什么想法？评论区见" |
| 置顶评论 | "看完这期，你学到了什么？回复告诉我" |
