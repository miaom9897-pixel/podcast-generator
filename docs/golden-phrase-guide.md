# 金句创作指南

## 📋 什么是好金句？

好金句 = 让人想截图 + 想转发 + 反复回味

---

# 一、金句的7种类型

## 1.1 对比式

**公式**：[大] vs [小] / [多] vs [少] / [期望] vs [现实]

**示例**：

- "他创造了200亿，拿了20万。"
- "16年干出中国第一，退休后什么都没有。"
- "前三个成了富翁，最后一个进了监狱。"

**写法技巧**：

- 数字越极端越好
- 落差越大越震撼
- 不需要解释，让对比说话

---

## 1.2 反常识式

**公式**：大多数人以为[X]，其实[反X]

**示例**：

- "在某些地方，能干是一种罪。"
- "努力不是解药，有时候是毒药。"
- "不是你不行，是你待错了地方。"

**写法技巧**：

- 找到受众的默认认知
- 直接反过来说
- 让人先愣一下，再"懂了"

---

## 1.3 规律总结式

**公式**：[现象]的规律是[XX]

**示例**：

- "问题不是你行不行，是你在什么位置。"
- "真正的润，不是肉身润，是脑子润。"
- "系统不会告诉你规则，但会惩罚你不懂规则。"

**写法技巧**：

- 从具体案例上升到普遍规律
- 用简单的话说深刻的道理
- 最好能让人当座右铭

---

## 1.4 因果揭示式

**公式**：因为[A]，所以[B]（但B让人意外）

**示例**：

- "因为太能干，所以进了监狱。"
- "因为创造了太多价值，所以被系统抛弃。"
- "因为贡献太大，所以成了威胁。"

**写法技巧**：

- 因果关系要出人意料
- 揭示隐藏的逻辑链
- 让人恍然大悟

---

## 1.5 选择式

**公式**：要么[A]，要么[B]（二选一）

**示例**：

- "要么换系统，要么被系统换。"
- "要么你定义规则，要么规则定义你。"
- "要么等别人给，要么自己去拿。"

**写法技巧**：

- 两个选项形成对立
- 暗示正确选项
- 制造紧迫感

---

## 1.6 时间线式

**公式**：[X岁][状态]，[Y岁][状态]，[Z岁][状态]

**示例**：

- "51岁起步，67岁巅峰，71岁入狱。"
- "年轻时怕错过机会，年老时怕失去本金。"
- "30岁觉得来日方长，50岁才知道时间不多了。"

**写法技巧**：

- 用时间点构建人生弧光
- 转折点要突出
- 让人看到自己的影子

---

## 1.7 丁元英式

**公式**：[深刻观点]，[补充说明]

**示例**：

- "这是文化属性决定的，不以任何人的意志为转移。"
- "规律不会因为你不喜欢，就对你网开一面。"
- "能人在传统文化里是没有出路的。"

**写法技巧**：

- 语气冷静，不带情绪
- 像在陈述事实，不是表达观点
- 有距离感但很有力量

---

# 二、金句创作公式

## 2.1 万能公式

```
[具体现象] + [规律总结]
```

**例子**：

- 现象：褚时健创造200亿只拿20万
- 规律：在某些系统里，你越能干，被剥削得越狠

---

## 2.2 反转公式

```
[常见认知] → 但 → [反转]
```

**例子**：

- 常见认知：努力就能成功
- 反转：在某些地方，努力会让你更危险

---

## 2.3 数字公式

```
[大数字] vs [小数字] = [结论]
```

**例子**：

- 大数字：200亿
- 小数字：20万
- 结论：10万倍的差距

---

## 2.4 问句公式

```
为什么[反常现象]？
因为[隐藏逻辑]。
```

**例子**：

- 反常现象：创造200亿的人只敢贪1000万
- 隐藏逻辑：他知道多拿一分都是罪

---

# 三、金句分布策略

## 3.1 位置与作用

| 位置 | 作用 | 金句类型 |
|------|------|----------|
| 开场第一句 | 抓注意力 | 数字对比/反常识 |
| 真相结尾 | 钉住观点 | 规律总结 |
| 拆解结尾 | 升华分析 | 丁元英式 |
| 意义延伸 | 唤起共鸣 | 选择式/时间线 |
| 收尾 | 让人回味 | 规律总结 |

## 3.2 分布密度

| 时长 | 建议金句数 |
|------|-----------|
| 30分钟 | 4-6个 |
| 60分钟 | 6-10个 |
| 120分钟 | 10-15个 |

**原则**：平均每10分钟1-2个金句

# 四、金句自检清单

写完金句后，用这个清单检查：

- [ ] 能不能用一句话说完？（不超过25个字）
- [ ] 有没有让人愣一下？
- [ ] 会不会想截图？
- [ ] 会不会想转发给朋友？
- [ ] 脱离上下文还能理解吗？
- [ ] 有没有AI味道？（太顺滑、太完整反而不好）
