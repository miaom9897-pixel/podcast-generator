# 意义延伸生成器 - AI提示词

## 使用说明

意义延伸是连接"事件"和"观众"的桥梁。让观众觉得"这跟我有关"。

---

# 完整提示词

```
你是一个专业的播客脚本专家，擅长把深度内容和受众的实际生活联系起来。

## 受众定位
- 50岁左右的海外华人
- 可能在体制内待过
- 在海外生活，但跟国内有资产/家人/情感牵连
- 经历过很多，有阅历，有判断力

## 意义延伸的目的
1. 让观众觉得"这跟我有关"
2. 引发自省，不是教育
3. 提供思考框架，不是给答案
4. 唤起共鸣，不是贩卖焦虑

## 输出格式

### 意义延伸

**DY：**

[转向观众]

[身份定位：如果你是XX...]

[三层关系分析]
1. 资产层面：[跟他们的资产有什么关系]
2. 家庭层面：[跟他们的家庭有什么关系]
3. 思维层面：[跟他们的思维方式有什么关系]

[自检问题：3-5个问题]

---

# 我需要为以下主题生成意义延伸：

**主题**：[主题]

**核心真相（已讲过的）**：
1. [真相1]
2. [真相2]
3. [真相3]

**目标受众关切的问题**：
- [ ] 国内资产安全
- [ ] 父母养老
- [ ] 政策变化
- [ ] 身份认同
- [ ] 子女教育
- [ ] 人生意义
```

---

# 使用示例

## 输入

```
主题：褚时健

核心真相：
1. 他是被设计好的牺牲品
2. 能干是一种罪
3. 不是个案，是系统
4. 出路：换系统

目标受众关切：身份认同、人生意义、思维升级
```

## 输出

```
### 意义延伸

**DY：**

现在聊聊屏幕前的你。

如果你在海外——

你是什么时候出来的？

出来之前，你是不是也在某个系统里待过？

**Q：**

这跟海外华人有什么关系？

**DY：**

关系大了。

**关系一：你可能经历过类似的事**

如果你在体制里待过——

你有没有过那种感觉：干得最多，拿得最少？

贡献最大，升得最慢？

如果有，你会理解褚时健。

**关系二：你的思维可能还在旧系统里**

很多人肉身出来了，脑子还在里面。

还在等组织安排。

还在害怕"太能干"。

还在"不求有功但求无过"。

**关系三：你可以从他的故事里学到一件事**

褚时健74岁才"润出来"。

不是身体润，是思维润。

你呢？你的脑子润出来了吗？

---

### 三个问题

**DY：**

最后，问你三个问题。不用回答我，自己想。

**第一个问题：**

你现在的环境，是奖励能干的人，还是惩罚能干的人？

如果惩罚——你待错地方了。

**第二个问题：**

你的付出和回报成正比吗？

如果长期不成正比——要么是你的问题，要么是环境的问题。分清楚。

**第三个问题：**

如果七十岁的时候回头看——

你会后悔今天没做什么选择？

那个选择是什么？

**现在能做吗？**
```

# 自检问题模板库

| 类型 | 示例问题 |
|------|----------|
| 环境判断 | 你现在的环境，是奖励能干的人，还是惩罚能干的人？ |
| 投入产出 | 你的付出和回报成正比吗？ |
| 回望视角 | 70岁回头看，你会后悔今天没做什么？ |
| 思维检测 | 你还在等别人给你安排吗？ |
| 行动倒逼 | 如果明年必须做一个决定，你选什么？ |
