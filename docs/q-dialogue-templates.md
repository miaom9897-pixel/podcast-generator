# Q角色话术库

## 📋 Q的定位

Q是观众的代言人，负责：

1. 问出观众心里的问题
2. 打断DY的长篇独白
3. 推进内容节奏
4. 增加可听性（两个声音比一个好）

---

# 一、Q的六种话术类型

## 1.1 困惑型

**作用**：表达听不懂，让DY解释

**模板库**：

- "等等，这是什么意思？"
- "我没听懂，你再说一遍？"
- "什么叫[术语]？"
- "这个我不太理解，能再解释一下吗？"
- "慢一点，我跟不上了。"

**示例**：

```
DY: 这就是国企的设计逻辑。
Q: 等等，什么叫国企的设计逻辑？
DY: 就是说...
```

---

## 1.2 追问型

**作用**：深挖一个话题，让DY展开说

**模板库**：

- "那他为什么要这样做？"
- "然后呢？"
- "后来怎么样了？"
- "那这件事是怎么发生的？"
- "所以呢？结果是什么？"
- "那问题来了..."
- "但是，为什么？"

**示例**：

```
DY: 他71岁被判无期。
Q: 那他为什么会出事？
DY: 官方说法是贪腐...
```

---

## 1.3 质疑型

**作用**：表达怀疑，让DY论证

**模板库**：

- "真的假的？"
- "有这么夸张吗？"
- "你是说他被人搞了？"
- "这听起来像阴谋论啊。"
- "有证据吗？"
- "你怎么知道的？"
- "不会吧？"

**示例**：

```
DY: 他一年给国家交200亿的税，自己只拿20万。
Q: 真的假的？200亿对20万？
DY: 真的，这是公开的数字...
```

---

## 1.4 总结型

**作用**：帮观众整理信息，确认理解

**模板库**：

- "所以你的意思是..."
- "也就是说..."
- "换句话说..."
- "等于说..."
- "我梳理一下..."
- "让我总结一下你刚才说的..."

**示例**：

```
DY: （讲了一大段分析）
Q: 所以你的意思是，他不是因为贪，是因为太能干？
DY: 可以这么理解。
```

---

## 1.5 连接型

**作用**：推进话题，连接到下一个部分

**模板库**：

- "那这跟我们有什么关系？"
- "那对海外华人来说呢？"
- "那我们能怎么办？"
- "那出路在哪？"
- "那现在呢？"
- "那后来发生了什么？"

**示例**：

```
DY: 这是褚时健的故事。
Q: 那这跟我们有什么关系？
DY: 关系大了...
```

---

## 1.6 情绪型

**作用**：表达情感反应，增加共鸣

**模板库**：

- "这也太惨了吧。"
- "离谱。"
- "这合理吗？"
- "听着就生气。"
- "这公平吗？"
- "不是吧？"
- "我服了。"

**示例**：

```
DY: 他创造了200亿，退休的时候什么都没有。
Q: 这也太惨了吧。
DY: 更惨的在后面...
```

# 二、Q的出场时机

## 2.1 必须出场的位置

| 位置 | 作用 | 推荐话术类型 |
|------|------|-------------|
| 开场后2-3分钟 | 打断开场节奏 | 困惑型/追问型 |
| 铺垫结束时 | 过渡到正题 | 追问型 |
| 每个真相抛出后 | 确认理解 | 困惑型/总结型 |
| 拆解结束时 | 过渡到下一真相 | 连接型 |
| 意义延伸开始时 | 引出与观众的关系 | 连接型 |

## 2.2 可选出场的位置

| 位置 | 何时用 | 推荐话术类型 |
|------|--------|-------------|
| DY讲了超过5分钟 | 避免太闷 | 任意类型 |
| 说了一个惊人数据 | 增加冲击 | 质疑型/情绪型 |
| 需要总结的地方 | 帮观众整理 | 总结型 |

---

# 三、Q的分布密度

## 3.1 按时长分布

| 时长 | Q出场次数 | 平均间隔 |
|------|-----------|----------|
| 30分钟 | 5-8次 | 4-6分钟 |
| 60分钟 | 8-12次 | 5-7分钟 |
| 120分钟 | 15-20次 | 6-8分钟 |

## 3.2 按节奏分布

| 时间段 | Q频率 | 原因 |
|--------|-------|------|
| 前15分钟 | 高（每3-5分钟） | 抓住节奏，让观众适应 |
| 中间部分 | 中（每5-8分钟） | 保持互动但不打断太多 |
| 后15分钟 | 低（每8-10分钟） | 让DY完整收尾 |

# 四、Q的语气指南

## 4.1 Q的人设

- 不是专家，是普通人
- 代表观众的视角
- 可以困惑、质疑、追问
- 不抢DY的风头
- 不发表自己的长篇观点

## 4.2 语气控制

| 情况 | 语气 |
|------|------|
| 困惑时 | 真诚的不理解，不是装傻 |
| 追问时 | 好奇，不是拷问 |
| 质疑时 | 友好的质疑，不是挑衅 |
| 情绪时 | 真实反应，不是表演 |

## 4.3 禁忌

| 禁忌 | 原因 |
|------|------|
| Q说太多 | 抢DY风头 |
| Q太聪明 | 失去代表观众的定位 |
| Q太频繁 | 打断节奏 |
| Q太少 | 变成单口播客 |
| Q的问题太假 | 观众能感觉出来是套路 |

# 五、Q话术速查表

## 5.1 按场景速查

| 场景 | 推荐话术 |
|------|----------|
| DY说了一个数据 | "真的假的？" / "有这么夸张吗？" |
| DY讲了5分钟以上 | "等等，我整理一下..." |
| DY说了一个专业术语 | "什么叫[XX]？" |
| DY讲完一个故事 | "那后来呢？" / "那这跟我们有什么关系？" |
| DY说了一个惨的事 | "这也太惨了吧" |
| DY讲完一个真相 | "所以你的意思是..." |
| 需要过渡到下一话题 | "那还有呢？" / "但是..." |

## 5.2 万能应急话术

当不知道Q该说什么时，用这些：

- "然后呢？"
- "为什么？"
- "真的吗？"
- "所以呢？"
- "等等，我没跟上。"

# 六、完整对话示例

```
DY: 褚时健出事，官方说是贪腐。1000多万。

Q: 1000多万，听起来不少啊。

DY: 但你想想，他给国家赚了多少？

Q: 多少？

DY: 16年，累计上千亿。他拿了多少？1000多万。差不到万分之一。

Q: 真的假的？这差距也太大了。

DY: 这就是国企的设计。你创造价值，系统拿走价值。

Q: 那他为什么还要拿那1000多万？

DY: 因为他67岁了，快退休了。退休之后呢？什么都没有。

Q: 等于说干了16年，最后白干？

DY: 可以这么说。一分钱的股权没有，一分钱的分红没有。

Q: 这也太惨了吧。

DY: 更惨的在后面...

Q: 还有更惨的？

DY: 他不是因为贪进的监狱。他是因为太能干进的监狱。

Q: 什么意思？你是说他被人搞了？

DY: 我没有证据说他被谁搞了。但我可以告诉你一个规律...
```
