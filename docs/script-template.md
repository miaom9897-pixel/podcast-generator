# 脚本结构模板

## 📋 模板说明

本文档定义了《元英视角》所有视频的标准脚本结构。

**核心原则**：统一框架，灵活内容。

---

## 🎬 脚本三段式结构

每个脚本必须包含三个明确分离的部分：

```
第一部分：制作信息（不念）
第二部分：完整脚本（念这个）
第三部分：辅助材料（不念）
```

# 第二部分：完整脚本

> **用途**：给AI配音、或自己录音用
> **是否念**：✅ 需要念

## 通用结构

所有类型的视频都遵循这个框架：

```
1. 【冲击开场】30秒-1分钟
   - 一个反常识的数据/事实/问题
   - 让观众想继续听

2. 【核心内容】15-18分钟
   - 人物类：3个规律（固定数量）
   - 时政类：3层分析（表象→原因→本质）
   - 每个规律/层之间有Q的提问过渡

3. 【海外华人连接】2-3分钟
   - 这件事跟你有什么关系？
   - 直接喊话受众

4. 【总结收尾】1-2分钟
   - 回顾核心观点
   - 留问题让观众思考
   - 固定收尾语
```

## 对话格式

```markdown
**DY：**

[丁元英的台词]

**Q：**

[提问者的台词]
```

# 不同类型的结构差异

## 类型A：人物类（资本往事/历史镜像）

```
开场：一个反常识的数据/事实
规律一：[人物的第一个规律]
规律二：[人物的第二个规律]
规律三：[人物的第三个规律]
海外华人：跟你有什么关系
总结：三个规律 + 三个问题
```

**固定3个规律**，不多不少。

## 类型B：时政类

```
开场：这周发生了什么 + 本质是什么
第一层：表象（发生了什么）
第二层：原因（为什么会这样）
第三层：本质（底层规律是什么）
第四层：影响（对你有什么影响）
海外华人：你该怎么办
总结：核心观点 + 留问题
```

## 类型C：事件分析复合类

```
开场：A事件 + B事件 = 说明什么
线索一：A事件分析
线索二：B事件分析
交叉：放在一起看说明什么
影响：对普通人/海外华人意味着什么
总结：核心观点 + 行动建议
```

# 时长控制

| 类型 | 目标时长 | 字数 |
|------|----------|------|
| 人物类 | 18-22分钟 | 7,000-8,500字 |
| 时政类 | 20-25分钟 | 8,000-10,000字 |
| 复合类 | 22-28分钟 | 8,500-11,000字 |

**原则**：宁短勿长。能18分钟说清楚的，不要拖到25分钟。

# 固定元素

## 固定开场语（可选用）

- "今天先算一道算术题。"
- "这周两件事，放在一起看，你就懂了。"
- "先说结论：XXXXX。"
- "今天不讲励志故事。励志故事你在抖音上能刷一百个。"

## 固定收尾语

```markdown
想明白了，就行动。
想不明白，就继续看下一期。
早晚能想明白的。
就怕你根本不想。

**我是元英，下期见。**
```

## 固定提问风格（Q的角色）

- 代表观众提问
- 不傻，但没DY看得深
- 会追问、会质疑
- 不是托儿，是真的想知道
